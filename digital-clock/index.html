The CSS3 Digital Clock uses many of the newest web technologies, such as CSS3 transforms and transitions, to create cool visual effects. Unfortunately, Internet Explorer does not yet support these technologies, and so you are probably seeing something less than stellar right now. If you switch to Chrome or Firefox, you'll not only see the clock in all it's glowing coolness, you'll probably have a much better overall experience when using the web.
