Fun with CSS-only bubbles

Hover over this paragraph for a quick and dirty tooltip using CSS3 transitions and some minimal Javascript. Hover over the bubbles below for a bit more useless WebKit fun.

The shadows, rounded corners and animation are all done with CSS, with a small bit of Javascript to trigger the bubble show/hide when hovering over the paragraph above. The rounding and shadows will work in most modern browsers, but you'll need WebKit to see the animations. The triangles are done with a variation of this technique.

See how cool these are? I had to make this text long enough to wrap because this technique is not perfect and the arrow doesn't look right with just one line of text.
